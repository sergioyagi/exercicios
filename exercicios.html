Guia Básico do GitHub

1. Criando uma Conta

Acesse GitHub e clique em "Sign Up" para criar sua conta.

2. Instalando o Git

Baixe e instale o Git a partir do site oficial: git-scm.com

Após a instalação, configure seu usuário e e-mail:

            git config --global user.name "Seu Nome"
            git config --global user.email "seuemail@example.com"

3. Criando e Clonando Repositórios

Para criar um novo repositório, clique em "New Repository" no GitHub e siga as instruções.

Para clonar um repositório existente, use:

            git clone https://github.com/usuario/repo.git

4. Trabalhando com Commits

Adicione e confirme alterações com os seguintes comandos:

            git add .
            git commit -m "Mensagem do commit"

5. Enviando para o GitHub

Envie suas alterações para o repositório remoto:

            git push origin main

Guia criado para ajudar iniciantes no GitHub.
